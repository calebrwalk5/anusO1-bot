  <h1>Commands</h1><br>
  !anus help: help menu <br>
  !anus random: random link or image <br>
  !anus coinflip: flips a coin<br>
  !anus meme: sends a random meme<br>
  !anus avatar: shows your avatar<br>

  !anus dababy: LESS GOO!!<br>
